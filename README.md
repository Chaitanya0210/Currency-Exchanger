# Currency Converter

A simple and responsive Currency Converter web application that allows users to convert currencies in real time using exchange rate data from a public Currency API.

## Features

- Convert between multiple currencies
- Dynamic currency dropdowns
- Automatic country flag updates
- Real-time exchange rate fetching
- Input validation
- Responsive and clean UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Currency API
- Font Awesome

## How It Works

1. Select the source currency.
2. Select the target currency.
3. Enter the amount.
4. Click **Get Exchange Rate**.
5. The application fetches the latest exchange rate and displays the converted amount.
